# Кто ты на шахматной доске? — Telegram-бот

Telegram-бот для MSM EduLeague × Sapiens Chess: тест из 10 вопросов, который
определяет шахматный стиль игрока — Тактик, Стратег, Универсал или Учёный
(либо Защитник, если несколько стилей набрали поровну баллов), — и ведёт
пользователя на регистрацию на турнир.

## Демо

![Демо бота](demo/demo.gif)

## Как это работает

1. Пользователь пишет `/start` → видит приветствие с баннером и кнопку
   «🎯 Начать тест».
2. Бот задаёт 10 вопросов с картинками, под каждым — 4 кнопки-варианта.
   Каждый ответ добавляет балл одному из 4 стилей игры.
3. После последнего вопроса бот считает, какой стиль набрал больше всего
   баллов, и показывает результат с коротким описанием и примером
   известного шахматиста этого стиля.
4. Под результатом — кнопки «✅ Хочу на турнир!» (ссылка на регистрацию)
   и «🔄 Пройти заново».

## Стек

- Python 3.11
- [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot) — взаимодействие с Telegram Bot API
- python-dotenv — хранение токена и настроек в `.env`

## Структура проекта

```
bot.py            обработчики команд и кнопок, вся логика бота
quiz_data.py       тексты вопросов, варианты ответов, описания результатов
images/            картинки вопросов и приветствия
requirements.txt   зависимости
.env.example        шаблон переменных окружения (без секретов)
```

Тексты вопросов и результатов вынесены отдельно от логики бота — поменять
формулировку вопроса можно, не трогая `bot.py`.

## Запуск локально

```bash
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env            # вписать свой токен бота
python3 bot.py
```

Токен бота получают у [@BotFather](https://t.me/BotFather) в Telegram.

## Технические детали

- Состояние пользователя (баллы по ходу теста) хранится в памяти процесса
  (`context.user_data`), что достаточно для короткого теста на 10 вопросов.
- Картинки вопросов читаются в память один раз при старте бота — для смены
  изображения нужно заменить файл в `images/` и перезапустить бота.
- Полный текст вариантов ответа показывается в подписи к вопросу, а сами
  inline-кнопки — это короткие иконки (🅐🅑🅒🅓), так как Telegram Bot API
  не позволяет управлять шириной или шрифтом inline-кнопок.
- Сетевые запросы к Telegram оборачены в повтор с задержкой на случай
  временных сбоев соединения.
